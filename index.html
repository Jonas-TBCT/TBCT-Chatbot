Chatbase is an AI chatbot builder that allows companies and other users to create their own chatbot trained on their specific data. Chatbase comes with a monthly subscription plan varying from $19/month to $399/month.

Flowise

Flowise AI is an open-source UI visual tool specifically designed for building Language Learning Model (LLM) apps. It is based on LangChain and offers a user-friendly drag-and-drop interface that simplifies the construction of complex LLM models.

Azure

Azure Bot Service is a cloud service provided by Microsoft Azure that enables developers to build, deploy, and manage intelligent bots that interact with users via various channels such as web, mobile apps, Microsoft Teams, Slack, Facebook Messenger, and more.
